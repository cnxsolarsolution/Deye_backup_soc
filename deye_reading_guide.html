Production (Solar)
กำลังไฟที่แผงกำลังผลิตตอนนี้
กำลังไฟฟ้าปัจจุบันจากแผงโซลาร์ (W หรือ kW) — ช่วงเที่ยงจะมีค่าสูงสุด ถ้าแดดจัด ตัวอย่างในรูป = 3.53 kW
⚡
Grid (การไฟฟ้า)
ไฟที่ดึงจากหรือส่งออกไปการไฟฟ้า
ลูกศรเข้า = กำลังซื้อไฟจากกริด
ลูกศรออก = ส่งขายคืนกริด (ถ้าระบบเปิดไว้)
ตัวอย่างในรูป = 19W (ดึงจากกริดนิดหน่อย)
🔋
Battery (แบตเตอรี่)
สถานะและทิศทางแบตเตอรี่
เปอร์เซ็นต์ (%) = ระดับแบตที่เหลือ (SOC)
ค่า W/kW = อัตราการชาร์จ/คาย
ในรูป 34% และ -2.91 kW = กำลังชาร์จเข้าแบต (เพราะ Solar ผลิตเหลือ)
🏠
Load (โหลดในบ้าน)
กำลังไฟที่บ้านใช้อยู่ตอนนี้
ปริมาณไฟทั้งหมดที่เครื่องใช้ไฟฟ้าในบ้านกำลังดึงอยู่ — รวมแอร์ ตู้เย็น ไฟ ฯลฯ ตัวอย่างในรูป = 374W
🔍 วิธีอ่านเครื่องหมาย บวก/ลบ ของ Battery
+
ค่าบวก (+) = แบตกำลังคาย (จ่ายไฟออกไปใช้)
−
ค่าลบ (−) = แบตกำลังชาร์จ (รับไฟเข้าเก็บ)

🔋 ระบบ Hybrid ทำงานอย่างไร?
ระบบ Hybrid คือการผสมผสาน 3 แหล่งพลังงาน — Solar + Battery + Grid — ให้ทำงานร่วมกันอย่างชาญฉลาด โดยมีลำดับความสำคัญดังนี้:
☀️ 1. ใช้ Solar ก่อนเป็นอันดับแรก
เมื่อมีแดด ระบบจะนำไฟจากแผงโซลาร์ไปใช้ในบ้านเป็นอันดับแรก
🔋 2. เหลือใช้ → ชาร์จแบต
หาก Solar ผลิตได้มากกว่าที่บ้านใช้ พลังงานส่วนเกินจะไปชาร์จแบตเตอรี่
⚡ 3. แบตเต็มแล้ว → ส่งกริด
เมื่อแบตเต็ม 100% ไฟส่วนเกินจะส่งออกขายคืนให้การไฟฟ้า (ถ้าตั้งค่าไว้)
🌙 4. ตอนกลางคืน → ใช้แบต
เมื่อไม่มีแดด ระบบดึงไฟจากแบตเตอรี่มาใช้ก่อน ถ้าแบตหมดเหลือที่ Reserve แล้วค่อยใช้ไฟกริด
